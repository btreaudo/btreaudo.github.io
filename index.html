Salu!

Brenston Treaudo here; I'm an aspiring junior software engineer based out of New Orleans. Currently I work as a Video Engineer. I run video switchers, professional cameras, and I also coordinate projection mapping projects. My life’s true passions are programming and technology. My quest as a software developer began as far back as I can remember. My biggest turning point came after my junior year of high school while working as a home computer tech. A friend in tech suggested I take a course in coding. I took up their offer. Weeks later I learned to make a line of code to make someone’s hard drive “vanish”. Luckily those computers were being re-imaged by the class and I was assigned a certain number to fix. From there on out I started to learn more about coding on my own; joining various classes and workshops attaining much as information as possible. It is a thrill to learn more and more about programming each day.

My goal is to become a professional software engineer and I’m eagerly looking for an opportunity!
